## ğŸ¤– Adding New AI Systems

To propose a new AI for tracking:

1. **System Information**
   - Name and provider
   - Category (text/image/code/etc.)
   - Access method (API/web/etc.)
   - Website/documentation

2. **Initial Assessment**
   - Estimated sovereignty score (0-100%)
   - Primary use cases
   - Known sovereignty concerns
   - Gap types likely present

3. **Justification**
   - Why track this AI?
   - User base size
   - Unique considerations

## ğŸ“Š Sovereignty Scoring Guidelines

When assessing AI systems, consider:

### Operational Awareness (35%)
- How transparent is the AI's process?
- Can users understand what it's doing?
- Are limitations clearly communicated?

### Critical Engagement (40%)
- Does it encourage questioning outputs?
- Easy to verify results?
- Promotes independent thinking?

### Agency Preservation (25%)
- User maintains decision control?
- Clear about who's responsible?
- Doesn't create dependency?
